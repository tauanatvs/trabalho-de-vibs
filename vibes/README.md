# vibes

Primeiros passos:

1- Como clonar o repositório na minha máquina, pelo vs code?

Abra um terminal no vs code e cole:

git clone https://github.com/liv3812/vibes.git

2- Após clonar o repositório, abra a pasta dentro do vs code e digite o comando:

git pull

(esse comando "puxa" todas as alterações que estão no repositório remoto para sua pasta local)

3- Para que suas alterações/adições apareçam no github, dê os seguintes comandos no terminal:

- git add.
- git commit -m "nome do seu commit"
- git push- u origin main


Atenção: sempre que for mexer no código, dê o comando git pull no terminal antes para puxar as modificações que seus colegas fizeram para dentro do seu computador!
